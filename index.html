Computer science is a vast field, and we explore 3 main branches of it:

Artificial Intelligence

Our newest branch, it focuses on creating, exploring and understanding Artificial Intelligence. We will learn to use various easy-to-understand tools created by Google, MIT, STEMpedia, etc. to create actual AI models that solve real-world problems. (No prior programming experience required)

Python (Basic and Advanced)

Learning text-based programming can enhance your critical thinking skills and give you the skillset to make programs and applications do various tasks. Here, we will explore the logical foundations of programming with a language known as Python. After you have a strong grasp on the foundations of Python, you can take your programming skills to the next level exploring logic gates, data analysis, the nuances of Python, etc. (Some python experience is preferable)

Game Development with Unity

Programming can be fun, too! Here, we will learn to create basic games and environments using a game engine called Unity. You will create many projects, learning the fundamental aspects of the application and logic in the process.
